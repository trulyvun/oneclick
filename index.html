What it touches

System areas modified

Each category groups related registry and service changes. The full diff is readable in the source.

Telemetry & Diagnostics

Disables DiagTrack, CEIP, Windows Error Reporting, and related scheduled tasks. Blocks telemetry endpoints via hosts file.

Background Services

Disables or sets to manual: SysMain, Windows Search indexing, Connected User Experiences, Xbox services, and other non-essential daemons.

Privacy Settings

Turns off advertising ID, activity history, location access for system processes, app diagnostics sharing, and feedback frequency.

Visual Overhead

Disables animations, transparency effects, and Aero peek. Reduces paint overhead on low-VRAM systems.

Power Plan

Switches to High Performance or Ultimate Performance. Disables CPU parking and USB selective suspend.

Network Stack

Sets TcpAckFrequency, disables Nagle's algorithm, adjusts auto-tuning. Targeted at latency, not throughput.

How it works

No magic.

It's a script. Here's exactly what happens when you run it.

01

Creates a restore point

Before touching anything, a System Restore snapshot is made. Roll back any time from Windows Recovery or the Settings app.

02

Applies registry changes

Registry keys are written to disable telemetry, adjust visual settings, and configure service startup types. All changes are in the script.

03

Stops and disables services

Non-essential Windows services are stopped and set to Manual or Disabled. Re-enable any of them from services.msc.

04

Applies network and power tweaks

Power plan switched via powercfg. TCP parameters written to the registry. Standard Windows configuration — no kernel patches.

05

Prompts for a restart

Some changes take effect on next boot. A restart prompt appears at the end. Skip it and restart manually if you prefer.

Requirements

Admin rights

Run as Administrator. The script modifies system-level registry keys and service states.

Windows 10 / 11

Tested on 22H2 and later. Some tweaks are no-ops on older builds but won't cause errors.

Read the script first

It's short and readable. Know what you're enabling or disabling. Don't run scripts blindly.

Some things may break

Disabling Windows Search breaks certain app search features. Disabling Xbox services may affect Game Bar. The restore point covers you.
